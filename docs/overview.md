# 🩺 drbiobit tools - Overview

`drbiobit tools` is a professional productivity suite designed for high-focus individuals. It combines a task manager, a Pomodoro timer, and a productivity analytics dashboard into a single, cohesive interface.

## Vision
The tool aims to provide a distraction-free environment where users can plan their day, execute focus sessions, and analyze their performance using data-driven insights.

## Core Modules

### 1. Task Management System
A comprehensive TODO app that goes beyond simple lists. It handles:
- **Prioritization**: High, Medium, and Low priority levels.
- **Categorization**: Customizable categories with color coding.
- **Temporal Tracking**: Due dates and overdue notifications.
- **Automation**: Intelligent recurring tasks that auto-generate the next instance upon completion.

### 2. Pomodoro Focus Timer
A specialized timer designed to prevent burnout and maintain high cognitive performance.
- **Standard Cycles**: Work $\rightarrow$ Short Break $\rightarrow$ Work $\rightarrow$ Short Break $\rightarrow$ Work $\rightarrow$ Short Break $\rightarrow$ Work $\rightarrow$ Long Break.
- **Customization**: Fully editable durations for each session type.
- **Logging**: Automatic tracking of focus duration and session counts.

### 3. Productivity Analytics
A data-driven dashboard that transforms raw activity logs into actionable insights.
- **Focus Metrics**: Total focused hours/minutes.
- **Efficiency Rate**: Completion percentage of daily tasks.
- **Historical Logs**: Daily, Weekly, and Monthly views of activity.
- **Gamification**: Achievement badges to motivate consistency.

## Design Philosophy
- **Sharp UI**: Zero border-radius for a professional, architectural look.
- **High Contrast**: Pure black background in dark mode to minimize eye strain and maximize focus.
- **Typography**: Use of `Plus Jakarta Sans` for clarity and `Orbitron` for the technical feel of the timer.
